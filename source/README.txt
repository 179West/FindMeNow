This folder contains the source code for the FindMeNow website.

The "form" folder contains code for the tab on the website that
displays a fillable form for lost property.

The "login" folder contains code for the tab on the website that
displays a login screen for users, as well as a registration
form for new users.

The "map" folder contains code for the tab on the website that
displays an interactive map containing clickable objects
representing reports.

The "resources" folder contains extraneous items that would be
displayed within the website itself.

The "settings" folder contains code for the tab on the website that
displays a series of options for managing customization of the
website, as well and for managing user accounts.
